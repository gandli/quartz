## 有趣的项目

1. [opencommit](https://github.com/di-sukharev/opencommit)

> 1 秒之内自动生成有意义的提交

![alt text](image.png)

```shell
npm install -g opencommit
```

`npx opencommit` 或 `bunx opencommit`

使用 `ollama`

```shell
git add .&& OCO_AI_PROVIDER='ollama' oco
```

```shell
oco config set OCO_LANGUAGE=zh_CN
```

配置文件`~/.opencommit`
